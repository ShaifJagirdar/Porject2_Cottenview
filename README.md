# Cotten Viewer

## Overview
Cotten Viewer is a simple and elegant web-based product showcase designed to display a cotton weave item with interactive elements like color selection, pricing, and rating. It provides a visually appealing product details page with a clean and modern UI.

## Features
- **Product Display**: Showcases a cotton weave product with a detailed view.
- **Color Selection**: Includes an option to toggle between different color variations.
- **Product Details**: Provides a description and highlights product specifications.
- **Pricing & Rating**: Displays the price along with a 5-star rating system.
- **Add to Bag Button**: Allows users to add the item to their shopping bag.
- **Responsive Design**: Fully responsive layout optimized for various screen sizes.

## Technologies Used
- **HTML5**: Structuring the web page.
- **CSS3**: Styling the interface.
- **Font Awesome**: Used for star rating icons.
- **CDN Image**: Hosted image for product display.

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/ShaifJagirdar/Porject2_Cottenview.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Porject2_Cottenview
   ```
3. Open `index.html` in your preferred browser.

## Usage
Simply open the `index.html` file in a web browser to view the product showcase. Users can explore the product details, pricing, and add items to the shopping bag.
